## 2141 우체국 🏣

처음에 모든 마을에 하나씩 우체국을 세워보고 비교해주었더니 시간초과가 났다.   
거리를 고려하지 않아도 되며, 인구의 누적 수만 고려해주면 된다는 것이 핵심인 것 같다.   

먼저 마을의 좌표를 기준으로 정렬해 준 후,   
왼쪽 인구의 누적수 >= 오른쪽 인구 누적 수 이고, 이 차이가 최소가 될 때가 답이 된다.   

풀이는 이분탐색 -> sol1.cpp 과    
그냥 단순 정렬 후 누적합 >= 절반이상 -> sol2.cpp   

두 가지로 나눠서 풀었다.
